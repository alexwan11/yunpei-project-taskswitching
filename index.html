Instructions

In the shape task:

If you see a circle, press the b key

If you see a rectangle, press the n key

In the color task:

If the stimulus is yellow, press the b key

If the stimulus is blue, press the n key

In the shape task (difficult):

If you see a circle in a square, press the b key

If you see a square in a circle, press the n key

In the color task:

If the stimulus is yellow, press the b key

If the stimulus is blue, press the n key

Note: For shape tasks, choose the color of the inner shape.

Shape Task: Press 'b' for circle, 'n' for rectangle.

Color Task: Press 'b' for yellow, 'n' for blue.
